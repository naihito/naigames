# NAIGAMES

NAIGAMESは、オリジナルゲームを **無料で遊べる**
ゲーム特化型のWebサイトです。

ゲーム開発(JavaScript)とWebサイト制作(Next.js)を通して、
**実践的なプログラミングスキル・設計・Git運用スキルの向上**を目的としています。

本プロジェクトは、**収益化を見据えたWeb制作スキルの向上**や
**将来的なポートフォリオ利用**を視野に入れています。

---

## NAIGAMESの位置付け

新プロジェクト「naisite」制作スタートに伴い
「NAIGAMES」は総合サイトである「naisite」に統合されます。

- **naisite**
  - 架空のWebサイトを集約・閲覧できる総合サイト
  - さまざまなデザイン・構成・UIパターン検証
  - ポートフォリオ・Webサイト制作用サンプルとして活用
- **NAIGAMES**
  - ゲーム特化型サイト
  - 実際にプレイ可能なゲーム公開
  - 総合サイトのnaisite内で運用

## Games

現在プレイ可能なゲーム：

- **ブロック無くし**
- **インベーダー無くし**
- **落ちもの無くし**
